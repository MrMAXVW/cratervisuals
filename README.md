# Crater Visuals (狂道影像) | Moving Image Portfolio

A high-impact, cinematic portfolio website built for **Crater Visuals**, a creative collective specializing in frantic and visceral moving images. The design emphasizes minimalism, dark aesthetics, and fluid motion to showcase professional video production works.

## 🌐 Project Overview
The website is a single-page immersive experience featuring:
- **Hero Reel**: A fullscreen, looping video background with high-impact typography.
- **Works Showcase**: A responsive grid featuring curated project highlights with YouTube integration.
- **Philosophy**: An "About" section defining the brand's creative identity ("Crater").
- **The Collective**: A minimalist list of team members with direct Instagram links.
- **Contact**: A high-contrast footer with a custom "Click to Copy" email feature and social links.

## ✨ Technical Highlights

### Frontend Stack
- **HTML5 & CSS3**: Structured with semantic tags and custom styling.
- **Tailwind CSS**: Used for rapid UI development and deep mobile optimization.
- **GSAP & ScrollTrigger**: Drives all animations, including scroll-based reveals.
- **Google Fonts**: Powered by 'Noto Sans HK' for clean multilingual typography.

### UI/UX Design
- **Custom Cursor**: A dynamic circular cursor for desktop users that scales and inverts colors.
- **Mobile Optimization**: Specifically tuned for mobile viewing (1.1x scaling for hero titles).
- **Smooth Loading**: Video metadata listeners to ensure a seamless fade-in.
- **Clipboard Integration**: Custom JS logic to copy the contact email.

## 🚀 Installation & Usage
Since the project relies on CDN-hosted libraries (Tailwind and GSAP), no local build process is required.

1. Clone the repo:
   git clone https://github.com/YourUsername/cratervisuals.git

2. Run it:
   Simply open index.html in any modern web browser to view the site.
